Contenido

Esta página es la tercera de una serie de páginas de práctica de formato HTML y CSS, aquí tenemos una barra de navegación, un banner principal, y tarjetas de selección que ofrecen un producto, también unos íconos de usuarios A, B, C, como para chat. Faltaría explorar cómo se coloca una barra de búsqueda, los grid, y ver que otros elementos UI son importantes.

El primer día aprendimos y practicamos el concepto de gradiente, el posicionamiento de los divs y elementos, y también un poco sobre la propiedad "display: flex", también manipulamos sombreado y entendimos sobre pseudoelementos ::before y ::after.

Hoy es el segundo día de pruebas, y fue dedicado a embellecer el formato del editor de código. Hay una especie de bitácora en un archivo *.txt sobre lo que intenté lograr. También intentamos añadir Dropdowns pero no funcionó, porque requiere ciertas bibliotecas de javascript, "popper", o algo así.

Hoy domingo terminamos este documento, lo dejamos hasta aquí porque bueno, varias razones... Entre ellas está que al parecer hay que aprender a usar a juro Bootstrap, y este documento no utiliza esas bibliotecas. Lo que hoy se logró fue añadir una lista de elementos con una interfaz más atractiva, se usaron "div" principalmente para todo, también se añadió un formulario final, solo para conocer las etiquetas HTML relacionadas y practicar un poco.
